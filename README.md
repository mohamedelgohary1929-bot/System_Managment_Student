#  Student Management System (C++)

##  Overview

A simple console-based Student Management System built using C++.

This system allows users to manage student records, including adding, removing, searching, and enrolling students in courses.

---

##  Features

* Add new students
* Remove students
* Search for a student by ID
* Display all students
* Enroll students in courses
* View student courses
* Sort students by GPA
* Save data to file

---

##  Technologies Used

* C++
* STL (vector, set, algorithm)
* File Handling (fstream)

---

##  OOP Concepts Used

* Inheritance
* Encapsulation
* Classes & Objects

---

## Project Structure

```
main.cpp
students.txt
```

---

##  How to Run

### 1. Compile

```
g++ main.cpp -o app
```

### 2. Run

```
./app
```

---

##  Data Storage

* Data is stored in `students.txt`
* Includes:

  * ID
  * Name
  * GPA
  * Courses

---

## Limitations

* No data loading from file at startup
* No GUI (console only)
* No update feature
